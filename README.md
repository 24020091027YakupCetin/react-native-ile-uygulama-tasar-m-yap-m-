# React Native ile Mobil Uygulama Tasarımı

Bu proje, React Native kullanılarak geliştirilmiş bir mobil uygulama çalışmasıdır. Proje kapsamında modern arayüz bileşenleri, kullanıcı deneyimi (UX) odaklı sayfalar ve dinamik tasarım öğeleri uygulanmıştır.

## 📱 Özellikler
* Temiz ve anlaşılır mobil arayüz tasarımı
* Sayfalar arası akıcı geçişler ve navigasyon yapısı
* Kullanıcı dostu ve modern bileşen (component) kullanımı

## 🛠️ Kullanılan Teknolojiler
* **Framework:** React Native
* **Dil:** JavaScript / TypeScript
* **Tasarım:** Flexbox Düzeni
